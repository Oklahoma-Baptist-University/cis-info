Setting Up Virtual Environments

You should create a folder for placing all your python environments in a place that is easy to remember; for example your home or Documents directory. Then, run the following command, where env-name is a name that is associated with environment you are creating (i.e., cis-3803-env):


python -m venv < env-name >


Next, navigate it into the directory just created:

cd < env-name >

The next step activates your environment and differs depending if you are running Windows or a Mac.

For windows, run the commands

cd Scripts
activate

For a Mac, run the commands

cd bin
source activate
